🎓 This is the repository for my personal portfolio, built with a focus on performance, cinematic animations, and a refined user experience.

## 🚀 Rodando localmente

```bash
# Clone o repositório
git clone https://github.com/RenanFrS/portfolio

# Instale as dependências (usa pnpm)
pnpm install

# Inicie o servidor de desenvolvimento
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) no browser.
